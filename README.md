BMTP语言，全称“**B**ASIC in **M**athematics **T**extbook of **P**EP”，中文全称“人教数学课本BASIC语言”，是人民教育出版社的《普通高中课程标准实验教科书数学3》第一
章《算法初步》中提到的一种“程序设计语言”。其语句形式和语法规则与BASIC语言类似。
